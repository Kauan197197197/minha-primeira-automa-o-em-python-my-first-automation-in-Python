## Descrição Geral
O objetivo deste projeto é desenvolver uma automação em Python utilizando PyAutoGUI para abrir o YouTube e realizar automaticamente uma pesquisa por aulas de Python. A proposta introduz conceitos de automação de interface de forma simples e prática.

---

## Funcionalidades Planejadas
- Abrir o navegador automaticamente.
- Acessar o site do YouTube.
- Inserir um termo de pesquisa definido pelo usuário.
- Executar o comando de busca.
- Permitir a modificação futura do termo pesquisado.

---

## Tecnologia Utilizada
O projeto utiliza exclusivamente a biblioteca:

- PyAutoGUI — responsável por simular movimentos do mouse, cliques e digitação.

Nenhuma outra ferramenta de automação de navegador é utilizada.

## General Description
